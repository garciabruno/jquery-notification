Demos

  • Default 
    $.createNotification({ content: 'Hola mundo!' })
  • Lower left 1 second 
    $.createNotification({
	horizontal: 'left',
	vertical: 'bottom',
	content: 'Hey hey Mar',
	duration: 1000
})
  • Image template 
    $.createNotification({
	content: '<img src="https://s-media-cache-ak0.pinimg.com/45x45/cd/80/5f/cd805f035fb20c990ddd31e0b04c5fde.jpg"> Hi, I\'m Trent Reznor'
})
  • Slow global fadeIn effect 
    $.extend($.notificationOptions, {
	fadeIn: 2000
})
$.createNotification({
	content: 'fadeIn is now set to 2000ms'
})
  • Reset global fadeIn effect 
    $.extend($.notificationOptions, {
	fadeIn: 400
})
$.createNotification({
	content: 'fadeIn is now set to 400ms'
})
  • Limit 2 (hit me many times!) 
    $.createNotification({
	content: 'Ho ho!',
	limit: 2,
	vertical: 'bottom'
})
  • Limit 2 with queue (hit me many times!) 
    $.createNotification({
	content: 'ba dum ts!',
	limit: 2,
	queue: true,
	horizontal: 'left'
})
  • Sticky 
    $.createNotification({
	content: 'I\'m not leaving punk',
	duration: null
})
  • Sticky, close on click 
    $.createNotification({
	content: 'I may leave, if you click on me',
	duration: null,
	click: function() {
		this.hide();
	}
})
  • Show only if window is on focus 
    $.createNotification({
	content: 'Change to a different tab for 3 seconds',
	duration: 3000
});

setTimeout(function(){
	$.createNotification({
		content: 'Hey, this showed up when you were on another tab',
		duration: 5000,
		vertical: 'bottom',
		showOnWindowFocused: true
	});
}, 1000);
  • Append to title 
    $.createNotification({
	content: 'Check your browser title',
	appendToTitle: ' - (' + Math.round(Math.random() * 10) + ')',
	duration: 3000
});
  • Append without restoring 
    $.createNotification({
    content: 'Check your browser title',
    appendToTitle: ' - (' + Math.round(Math.random() * 10) + ')',
    duration: 3000,
    restoreTitle: false
});
  • Set element background 
    $.createNotification({
    content: 'I\'m blue da ba dee da ba die',
    duration: 3000,
    background: '#0C6395'
});
  • Set element text color 
    $.createNotification({
    content: 'Everything went OK',
    duration: 3000,
    color: '#3CB500'
});

Installation

Include script after jQuery

<script src="jquery.notification.js"></script>

Include stylesheet

<link rel="stylesheet" href="notification.css">

Copy images (using same path of stylesheet)

Usage

$.createNotification(options)

Options

click

Click callback

content

Notification content

duration

On screen duration in milliseconds, set it to 0 for sticky, default 5000

fadeIn

Fade in effect duration in milliseconds, default 400

fadeOut

Fade out effect duration in milliseconds, default 400

limit

Visible limit per board, when reached notifications wont be created, default false

queue

If queue is active and limit is reached notifications will be shown when other hides, default false

slideUp

Slide up effect duration in milliseconds, default 200

horizontal

Horizontal alignment, default right

vertical

Vertical alignment, default top

showOnWindowFocused

Notification will only show if the window has focus, otherwise it will queue until focus is restored, default false

appendToTitle

Append a string to the browser title for the duration of the notification, default is not set

restoreTitle

Restore the browser title to it's original content, default true

background

Set element background property, default is set by css file

color

Set element text color property, default is set by css file
